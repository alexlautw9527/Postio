# 歡迎使用 Postio 👋

**Postio** 把 Markdown 轉成漂亮的社群純文字，貼到 Threads 或任何平台都好看。

*Write once. Publish anywhere.*

## 快速上手

1. 把這裡的內容整段換成你的 Markdown
2. 預覽區立刻出現 Threads 版本，超過 500 字會自動分段成串文
3. 逐篇按「複製」，依序貼到 Threads 就完成

## 支援的語法（原稿怎麼寫，預覽就怎麼變）

### 這是小標題

**這是粗體 Bold 123456789**（英文字母與數字才看得出變化）

*這是斜體 Italic* （只有英文字母才看得出變化）

`code is monospace 123456789`（等寬字，英文字母與數字才看得出變化）

[這是連結](https://example.com)

- 這是清單
    - 縮排兩格變巢狀
        - 最多到第三層
- [x] 這是完成的待辦
- [ ] 這是未完成的待辦

> 這是引用

## 自動分段

超過 500 字會自動切成串文，每篇結尾加上編號，照順序逐篇複製、貼上就完成。分段會挑段落邊界下刀，不切斷句子，標題與清單也不會被拆開。

也可以自己決定分割點：在要切開的地方單獨放一行 ===（前後留空行），就會強制換篇，下一行就是示範。

===

這一篇就是被 === 切出來的，編號也跟著更新了。祝您使用愉快！
